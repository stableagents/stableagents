Getting Started

Installation

StableAgents requires Python 3.9 or higher. You can install it using pip:

pip install stableagents-ai

For development:

git clone https://github.com/yourusername/stableagents.git
cd stableagents
pip install -e .

Quick Start

Here's a simple example to get you started with StableAgents:

from stableagents import StableAgents

# Initialize with your preferred configuration
agent = StableAgents(
    enable_self_healing=True,
    enable_logging=True
)

# Configure AI provider
agent.set_api_key('openai', 'your-api-key')
agent.set_active_ai_provider('openai')

# Generate text with the agent
response = agent.generate_text("Analyze the performance of our product in Q1")
print(response)
Tip
Store your API keys securely in environment variables or a configuration file that's not committed to version control.

API Reference

Authentication

All API requests require authentication using an API key.

POST
/v1/auth/token

Generate a new authentication token.

Parameter
Type
Description
api_key
string
Your StableAgents API key
expires_in
integer
Token expiration time in seconds (default: 3600)

Text Generation

Generate text responses using the configured AI provider.

POST
/v1/generate

Generate a text response based on the provided prompt.

Parameter
Type
Description
prompt
string
The input text prompt
provider
string
AI provider to use (optional, defaults to configured provider)
max_tokens
integer
Maximum number of tokens to generate (default: 1024)
temperature
float
Sampling temperature (default: 0.7)

Memory API

Store and retrieve information from the agent's memory.

POST
/v1/memory/store

Store information in the agent's memory.

Parameter
Type
Description
key
string
Unique identifier for the memory item
value
object
The data to store
ttl
integer
Time to live in seconds (optional)
GET
/v1/memory/retrieve/{key}

Retrieve information from the agent's memory.
